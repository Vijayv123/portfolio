# Vijay V Portfolio Website

A modern React + Vite portfolio website with project cards, profile photo, social links, and a working FormSubmit contact form.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Contact form

The form uses FormSubmit and sends messages to `vijayvraj2000@gmail.com`.
On first submission, FormSubmit may ask you to verify your email.

## Update project links

Open `src/App.jsx` and replace each `demoUrl` and `codeUrl` with your real deployed project and GitHub repository links.
